# 📊 Análisis de Inmigrantes Brasileños en Canadá

Este proyecto tiene como objetivo analizar el perfil y la evolución de los inmigrantes brasileños en Canadá, utilizando datos históricos y técnicas de análisis exploratorio en Python.

🔗 Repositorio:  
https://github.com/YoannaHS/analisis-inmigrantes-brasil

---

## 📌 Análisis realizado

La análisis fue desarrollada en Jupyter Notebook:  
🔗 https://github.com/YoannaHS/analisis-inmigrantes-brasil/blob/main/analisis_inmigrantes_brasileros.ipynb

Se exploraron tendencias de inmigración a lo largo de los años, distribución por países de América del Sur y visualizaciones para identificar patrones relevantes sobre la migración brasileña.

---

## 🗂️ Base de datos

La base de datos utilizada se encuentra en el siguiente archivo:

🔗 https://github.com/YoannaHS/analisis-inmigrantes-brasil/blob/main/inmigrantes_canada.csv

Este dataset contiene información sobre el número de inmigrantes de diversos países hacia Canadá, segmentados por año.

### Variables principales:
- País de origen  
- Año  
- Cantidad de inmigrantes
